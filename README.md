##  Requirements
Make sure you have:
- [Node.js](https://nodejs.org/) installed on your system

---

##  How to Run

Clone the repository and run the following commands:

```bash
git clone https://github.com/Riegooo/expense-tracker.git
cd expense-tracker
npm install
```

Run the following commands to start the game:
```bash
node index.js
```
